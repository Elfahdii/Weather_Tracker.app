# Weather\_Tracker.app

An app to track wheather

\# 🌦️ Weather Tracker



\## Overview

Weather Tracker is a Streamlit web application that allows users to record, store, search, and analyze weather observations. All observations are saved in a CSV file, making it easy to keep historical weather records and perform simple weather analysis.



\---



\## Features



\- Record new weather observations

\- Store weather data in a CSV file

\- View all recorded observations

\- Search observations by date

\- View weather statistics

&#x20; - Average temperature

&#x20; - Minimum temperature

&#x20; - Maximum temperature

&#x20; - Most common weather condition

\- Compare weather statistics between two different years

\- Predict tomorrow's weather using historical observations

\## How It Works



1\. Users record daily weather observations.

2\. The application stores each observation in a CSV file.

3\. Users can search observations by date.

4\. Users can view all recorded weather data.

5\. Users can view temperature statistics and the most common weather condition.

6\. Users can compare weather data between different years.

7\. The application predicts tomorrow's weather based on previous observations.
